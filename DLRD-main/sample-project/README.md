# Sample Todo project

A tiny multi-layer project used to demo the local Data-Lineage and Retro-Documentation dashboard.

- `api/` — request routing (API layer)
- `services/` — business logic (Service layer)
- `data/` — models and persistence (Data layer)
- `utils/` — shared helpers (Utility layer)
- `ui/` — a presentational component (UI layer)
